# GTA-editor

## Welcome to the GTA-editor documentation
The Generic Tool for Automation Editor is an editor module which allows the creation, modification, execution and results reporting of test Procedures on Ground Test Means.

This documentation provides information about the essential features of the GTA-editor source code for developers.
